单目3D目标检测项目 (Monocular 3D Object Detection)
====================================================

项目概述：
本项目实现了基于KITTI数据集的单目3D目标检测，包含多种深度学习架构、
完整的实验评估体系、数据分析和可视化工具。

核心特性：
 4种模型架构：Simple CNN, ResNet, Attention (CBAM), Transformer
 集成模型：结合多个模型的预测
 交叉验证：5折交叉验证评估模型泛化能力
 特征工程：特征重要性分析和特征探索
 降维聚类：PCA, t-SNE, K-Means, DBSCAN
 全面评估：AP, IoU, MAE, RMSE, FPS等多种指标
 数据可视化：Jupyter notebook交互式分析
 完整文档：详细的项目报告和使用说明

快速开始：
1. 安装依赖：pip install -r require.txt
2. 下载KITTI数据集并放入data/kitti/training/
3. 训练模型：python main.py
4. 评估模型：python evaluate.py
5. 数据分析：jupyter notebook data_exploration.ipynb

详细文档：
- 快速开始：QUICK_START.md
- 完整文档：README.md
- 项目报告：PROJECT_REPORT.md

模型架构说明：
1. Simple CNN (2.5M参数)：轻量级基线模型
2. ResNet (8.5M参数)：残差连接深度网络
3. Attention (3.2M参数)：CBAM注意力机制
4. Transformer (12.8M参数)：CNN+Transformer编码器
5. Ensemble：集成多个模型的预测

评估指标：
- Average Precision (AP@0.5, AP@0.7)
- Mean Absolute Error (MAE)
- Root Mean Square Error (RMSE)
- Inference Speed (FPS)
- 3D IoU / BEV IoU

数据分析功能：
- 数据分布统计
- 特征相关性分析
- PCA降维可视化
- t-SNE非线性降维
- K-Means聚类
- DBSCAN密度聚类
- 特征工程和重要性分析

实验设置：
- 训练/验证分割：80/20
- 交叉验证：5折
- 批量大小：4
- 学习率：1e-4
- 优化器：Adam
- 早停机制：patience=8
- 梯度裁剪：max_norm=5.0

技术栈：
- PyTorch 2.0+
- NumPy, Pandas
- Matplotlib, Seaborn
- Scikit-learn
- OpenCV
- Jupyter Notebook
